# ステップメモ

このアプリはメモ帳アプリです。
特徴としては、メモの形式が段階的にメモを作成することができます。

例) 勉強 
        - 国語
                - 漢字
                - 文学
        - 数学
                - １次関数
                - 2次間数
                - 図形
        - 社会 
                - 歴史
                - 地理
        - 理科
                - 植物
                - 科学

## 機能

* 親(Root)メモの作成
* 子メモの作成
* メモ編集
* メモ削除

## 保存内容
* メモ帳本文
* 履歴(保存ボタンをタップしたタイムスタンプを記録
